Glassmorphism 设计原则

最后更新: 2025年11月23日 | 作者: 你的昵称

“玻璃拟物”风格的核心在于 **透明度、多层级和色彩柔和**。它通过背景模糊和阴影来模仿玻璃卡片的质感，非常适合作为你个人工作区的 UI。

  • **背景模糊 (Blur)：** 使用 CSS 的 `backdrop-filter: blur(10px)`。
  • **多层级：** 通过 Z 轴堆叠卡片，制造深度感。
  • **边框透明：** 使用半透明的边框来模拟玻璃边缘的光泽。
